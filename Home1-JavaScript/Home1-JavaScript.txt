1.Const-константа. Переменную, объявленную через const, можно определить один раз и нельзя поменять. Может принимать любой тип данных.
Переменную, которая объявлена через let, можно переопределять несколько раз. Принимает любой тип данных.
Переменную, которая объявлена через var, такая же как и через let, но с другой областю видимости.

2. Var устаревшая форма объявления.
У переменной var другая область видимости, нежели у let. 
Она будет видна с любой точки кода (но если она будет в другой функции, то она будет видна только там). 
Так же var является глобальной и ее можно объявить больше одного раза в одном блоке.
